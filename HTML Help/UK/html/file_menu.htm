File Menu

New game…

Select this option to begin a new game of MONOPOLY CD-ROM. If a game is in progress, you will be given a chance to save the current game before starting a new one.

Open game…

Select this option to load a previously saved game of MONOPOLY CD-ROM. The “Open” dialog box will appear, and you may select which saved game you want to open.

Save game…

Select this option to save a game currently in progress. Saving allows you to continue a game at a later time. This option is only available for a game played on a single computer, as opposed to games played over the Internet or on a network. When the Save game dialog box appears, enter a filename for the saved game.

Exit

This option allows you to quit MONOPOLY CD-ROM. You will be given a chance to save the game in progress.
