Options Menu

Custom game rules…

MONOPOLY CD-ROM is played according to the traditional rules of MONOPOLY. However, it is possible to alter certain aspects of the game by tweaking the Custom game rules. To change any of the default options, select Custom game rules. A dialog box appears. Simply click on any of the options that you wish to change. If you are playing MONOPOLY CD-ROM over the Internet or on a Network, all remote players get a chance to change or agree to any options you have selected. This process continues until all changes have been agreed upon. For more information on how to begin an Internet or Network game, see Starting a Game.

International options…

MONOPOLY CD-ROM allows you to compete against players in other countries over the Internet. As the rules tend to differ slightly from country to country, the International options menu allows you to incorporate some of these different rules into your game. For example, under normal circumstances, only your standard Chance and Community Chest cards are used in a game. Click on one or more countries' decks to add them to your game. Click on an already selected deck to remove it from your game. To choose any of the rule options, click on that rule. Click on an already selected rule to remove it from the list.

Digital sound effects

Select this option to turn the game’s sound effects on or off. A tick opposite this option indicates that sound effects are currently switched on (this is the default setting).

Background MIDI music…

Select this option to call up a dialog box that allows you to turn the game’s background music on or off, change the MIDI output device, and arrange the sound track in the order that you prefer. A tick opposite this option indicates that the game’s background music is currently switched on.

Animation display options…

Select this option to turn selected 3D animations that run in the centre of the game board on or off. Each option refers to an animation or set of animations which you can switch on or off with a click of the mouse. All animations are set to play by default; to switch any of them off, simply click on them.

Computer AI options…

This option allows you to alter some of the playing characteristics of your computer-simulated opponents. Select Computer AI options and a dialog box will appear. You may switch any feature on by clicking on the empty box opposite. You may also cancel a selection by clicking on a feature that is currently selected.

Game board options…

This option allows you to set some game defaults. Select Game board options and a dialog box will appear. To select a feature, click on the empty box opposite. To cancel a selection, click once again on a currently selected feature.

Selecting the Spinning Token features will cause the current player's token to rotate on the board, in the player area, or both, during that player’s turn. Getting the computer to draw a white deed on mortgaged property will make it easier to remember which properties are mortgaged. Graying out vacant or mortgaged properties will make it easier to identify these properties during a game.

Animation background…

This option allows you to customize the backdrop of the token animation clips. When you select Animation background, a dialog box appears enabling you to choose an animation background file. Select or type in a background filename, and then click on OK. The new background will be displayed whenever a token animation is displayed.

Save options as default

This option will save all of your current game options settings, and make them the default settings. The next time you start MONOPOLY CD-ROM, these settings will be automatically loaded. In future games, if you change the settings again, you will need to choose this option again to save them.
