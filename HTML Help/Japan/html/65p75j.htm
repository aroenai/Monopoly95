ゲームの開始

MONOPOLY CD-ROM ゲームの開始は、一つのコンピュータでプレイする場合も、インターネットを使ってプレイする場合も、とても簡単です。新規ゲームを開始すると、ゲーム ボードの左側にプレイヤー追加用のボタンが表示されます。[プレイヤーの追加 #1] ボタンをクリックすると、ダイアログ ボックスが表示されます。ゲームに参加するプレイヤーの種類を選択してください。プレイヤーの名前を入力したら、使用する駒を選択します。必要に応じてプレイヤーを追加してください。インターネットまたはネットワークを使ってプレイする場合は、接続に必要な情報を用意する必要があります。

MONOPOLY CD-ROM ゲームは、一台のコンピュータ、またはインターネットまたはネットワーク上の複数のコンピュータを使って 6 人までのプレイヤーでプレイできます。インターネットまたはネットワークを使ってプレイする場合、一人のプレイヤーのコンピュータが「ホスト」となり、他のコンピュータを使っているプレイヤー (リモート プレイヤー) は、その「ホスト」コンピュータに接続します。ホストはリモート プレイヤーがホストに接続するのを待ちます。リモート プレイヤーは [インターネット/ネットワーク上のプレイヤー] をクリックし、ホストの IP アドレスを入力して接続します。

ホストに接続した場合、ゲームのすべてのルールはホストで設定されているルールになります。ルールは変更できますが、変更するには、参加者全員の承諾が必要になります。

IP アドレスとは、インターネット上でコンピュータを特定するために使用する、各コンピュータに設定されている番号のことです。使用しているコンピュータの IP アドレスがわからない場合は、インターネットの管理者やサービス プロバイダなどにお問い合わせください。
