トラブルシューティング

Q: MONOPOLY CD-ROM ゲームをハードディスクにインストールできないのですが？

A: ディスク圧縮のユーティリティを使用している場合は、ディスクの空き容量が実際に存在する空き容量より多く見える場合があります。不要なファイルなどを削除してディスクの空き容量を増やしてください。

Q: MONOPOLY CD-ROM ゲームをロードできないのですが？

A: MONOPOLY CD-ROM を再度インストールしてみてください。インストールするときに WinG と Win32s が選択されていることを確認してください。もし、問題が解決されない場合、Windows を再起動し、動作中の他のアプリケーションを終了してください。

Q: インターネットまたはネットワーク上で MONOPOLY CD-ROM ゲームをプレイしているときにゲームが遅くなるのですが？

A: 他のコンピュータでプレイしているプレイヤーが 3D ムービーをオンにしていると、3D ムービーをオフにしているコンピュータでは、ムービーが再生される間待たなければならないためです。

Q: MONOPOLY CD-ROM ゲーム中にサウンドが鳴らないのですが？

A: まず最初に、[オプション] メニューの [サウンド効果] 設定にチェックマークがあるかどうかを確認してください。チェックマークがない場合、このオプションをクリックしてサウンド効果をオンにしてください。これでもおかしい場合、Windows でサウンドが正しく設定されているかどうかを確認します。Windows のバージョンによって名前が異なる場合がありますが、基本的な操作は同じです。コントロール パネルで [サウンド] をダブルクリックします。ここで各種の Windows サウンド効果を設定できます。サウンド効果が設定できない場合は、Windows サウンド ドライバが正しくインストールされていません。サウンド ドライバをインストールしなおしてください。サウンド効果が設定できるのに音が聞こえない場合は、ボリューム、スピーカーの接続などに原因があると思われます。これらの設定を確認してください。

Q: MONOPOLY CD-ROM ゲームで MIDI ミュージックが再生されないのですが？

A: まず最初に、[オプション] メニューの [バックグラウンド MIDI ミュージック] 設定にチェックマークがあるかどうかを確認してください。チェックマークがない場合、このオプションをクリックして MIDI ミュージックをオンにしてください。これでもおかしい場合、Windows に MIDI ドライバが正しくインストールされているかどうかを確認します。コントロール パネルの [MIDI マッパー] ユーティリティを使用すると、インストール情報を確認できます。MIDI をテストするには、メディア プレーヤーを開いて [デバイス] メニューから [MIDI シーケンサ] を選択します。canyon.mid ファイルを選択して再生してみてください。音が聞こえない場合は、Windows サウンド ドライバをインストールしなおしてください。これでも解決しない場合は、ボリューム、スピーカーの接続などに原因があると思われます。これらの設定を確認してください。

Q: どうしても問題が解決できない場合はどうすればいいでしょう？

A: README ファイルの情報を参照してみてください。それでも問題が解決できない場合は、マニュアルに書かれているテクニカル サポートまでお問い合わせください。
