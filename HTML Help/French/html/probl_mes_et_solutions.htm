Problèmes et solutions

Q: Pourquoi ne puis-je pas installer MONOPOLY CD-ROM sur mon disque dur?

R: Il se peut que vous utilisiez un utilitaire de compression de disque dur. Ces utilitaires peuvent altérer l’espace disponible pour l’installation du jeu MONOPOLY CD-ROM. Essayez de faire de la place sur votre disque en eliminant des dossiers qui ne sont pas nécessaires.

Q: Pourquoi ne puis-je pas charger MONOPOLY CD-ROM?

R: Vous devez sans doute mettre à jour vos logiciels WinG et Win32s avec les versions qui se trouvent sur le disque du jeu MONOPOLY CD-ROM. Réinstallez le jeu, cette fois en sélectionnant WinG and Win32s pour l’installation. Si vous avez toujours des problèmes, éteignez et rallumez votre ordinateur et fermez toutes les applications en cours d’exécution.

Q: Pourquoi, quand je joue au MONOPOLY CD-ROM sur l'Internet ou sur un réseau, le jeu fait quelquefois une pause et/ou le pointeur de ma souris se transforme en sablier?

R: Il se peut que les joueurs sur les sites à distance aient choisi de passer des animations en 3 dimensions alors que vous les avez dé-sélectionnées. Attendez qu'ils aient fini de passer l'animation avant de reprendre le jeu.

Q: Pourquoi MONOPOLY CD-ROM ne joue t-il pas d'effets sonores?

R: Tout d'abord, vérifiez le menu Options et en particulier l'option Effets sonores numérisés. Il devrait y avoir une coche en face de cette option. Si ce n'est pas le cas, cliquez sur l'option pour activer les effets sonores. Si vous n'entendez toujours pas d'effets sonores, vérifiez que votre installation Windows actuelle supporte les sons. Si ce n'est pas le cas, essayez de les mettre en place. Il se peut que le nom des fichiers spécifiques soit légèrement différent selon la version de Windows que vous possédez mais le processus est le même pour toutes les versions. Faites apparaître le Panneau de Configuration et cliquez deux fois sur l'icône Son. Cela vous permettra de tester divers effets sonores Windows. Si vous n'arrivez pas à sélectionner les fichiers d’effets sonores, il est probable que les pilotes son Windows ne sont pas installés ou qu'ils ont été installés incorrectement. Réinstallez les pilotes son à partir du disque d'installation de votre carte son. Si vous arrivez à sélectionner les effets sonores mais si vous ne les entendez pas, il se peut que le volume soit réglé trop bas, que les hauts-parleurs ne soient pas alimentés en électricité ou qu'il y ait un mauvais contact. Si vous n'arrivez toujours pas à résoudre le problème, veuillez contacter notre téléphone rouge technique au numéro indiqué dans le manuel MONOPOLY CD-ROM.

Q: Pourquoi MONOPOLY CD-ROM ne joue-t-il pas de musique MIDI?

R: Tout d'abord, vérifiez le menu Options et en particulier l'option Musique de fond MIDI. Il doit y avoir une coche en face de cette option. Dans le cas contraire, cliquez sur cette option pour avoir la musique MIDI. Si vous ne pouvez toujours pas entendre de musique, vérifiez que vous avez installé un programme pilote MIDI pour Windows. Vous l'avez sans doute fait au moment de l'installation des programmes pilote de votre carte son. Vous pouvez rechercher les informations d'installation dans la rubrique MAPPEUR MIDI du Panneau de Configuration. Pour tester votre capacité de reproduction MIDI, sélectionnez Diffuseur de Médias puis Séquenceur MIDI au menu Périphériques. Sélectionnez le fichier canyon.mid. Si vous n'entendez pas de son, essayez de réinstaller les programmes pilote Windows de votre carte son. Si cela n'a pas d'effet, vérifiez le volume des hauts-parleurs, les connexions et les câbles électriques des hauts-parleurs. Si vous avez toujours des problèmes, veuillez contacter notre téléphone rouge technique au numéro indiqué dans le manuel MONOPOLY CD-ROM.

Q: Que puis-je faire si rien ne semble fonctionner?

R: Quittez Windows, éteignez votre ordinateur, puis rallumez-le. Après avoir initialisé Windows, vérifiez qu’il n’y a pas d’autres applications en fonctionnement. Si certaines aplications sont utilisées fermez -les avant d’initialiser le jeu MONOPOLY CD-ROM. Vérifiez aussi les informations de dernière minute dans le fichier “Lisez-moi” pour voir si certains commentaires peuvent s’appliquer à votre situation. Finalement, appelez nos ingénieurs au numéro que vous trouverez dans le manuel. Vous pouvez aussi consulter la section Aide du manuel pour plus d’information sur l’Aide électronique en ligne.
