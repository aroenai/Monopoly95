Messagerie/bavardage électronique

Pour bavarder avec un joueur à distance alors que vous êtes en train de faire une partie de MONOPOLY CD-ROM sur Internet ou sur un réseau, cliquez sur le pion du joueur à distance dans sa zone de jeu sur votre écran. Une boîte de dialogue apparaîtra qui vous permettra d’entrer un message. Ce message peut être ‘chuchoté à l’oreille’ de celui à qui il est destiné ou bien on peut ‘crier’ ce même message pour que tous les joueurs puissent le voir.
