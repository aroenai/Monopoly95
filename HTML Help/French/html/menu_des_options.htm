Menu des Options

Règles du jeu “sur mesure”…

Le MONOPOLY CD-ROM se joue de la même manière que le MONOPOLY traditionnel. Cependant il est possible de changer certains aspects du jeu en utilisant les règles du jeu sur mesure. Afin de changer n’importe laquelle des options “par défaut”, sélectionnez Règles du jeu sur mesure. Une boîte de dialogue apparait. Cliquez simplement sur n’importe laquelle des options que vous désirez changer. Si vous faites une partie de MONOPOLY CD-ROM sur Internet ou sur un Réseau, tous les joueurs à distance ont également la possibilité de changer, ou d’acccepter les options que vous avez choisies. Ce processus continue jusqu’au moment où tous les changements ont été acceptés par chaque joueur. Pour plus d’information en ce qui concerne la manière d’initialiser une partie sur Internet ou un Réseau, reportez-vous à Démarrage d'une partie.

Options Internationales…

Le jeu MONOPOLY CD-ROM vous permet de vous mesurer à des joueurs d'autres pays sur Internet. Comme les règles sont légèrement différentes d'un pays à l'autre, le menu Options internationales vous permet de jouer avec certaines de ces différences. Par exemple, dans des circonstances normales, seules les cartes Chance et Caisse de Communauté standard sont utilisées dans une partie. Mais vous pouvez cliquer sur les jeux de cartes d'un ou plusieurs pays pour les ajouter à votre partie. Et bien entendu, cliquez sur un jeu déjà sélectionné pour l'éliminer de votre partie. Pour choisir une des règles internationales, cliquez sur cette règle. Cliquez sur une règle déjà sélectionnée pour l'enlever de la liste.

Effets sonores numériques

Choisissez cette option afin de sélectionner ou d’annuler les effets sonores. Une coche en face de cette option indique que les effets sonores ont été sélectionnés.

Musique de fond…

Choisissez cette option afin de sélectionner ou d’annuler le fond sonore du jeu, ou bien pour changer le périphérique MIDI ou pour organiser le fond musical selon vos préférences. Une coche en face de cette option indique que la musique de fond est sélectionnée.

Options des films d'animation

Choisissez cette option afin de sélectionner ou d’annuler les animations en trois dimensions qui passent au centre de l'écran. Chaque option renvoie à un film d'animation ou un groupe de films que vous pouvez sélectionner et dé-sélectionner en cliquant dessus avec la souris. Tous les films sont sélectionnés par défaut. Pour les dé-sélectionner, cliquez dessus.

Options d’Intelligence Artificielle (IA)…

Cette option vous permet de déterminer des caractéristiques bien particulières pour vos adversaires simulés par ordinateur. Quand vous sélectionnez Options d’Intelligence Artificielle, une boîte de dialogue apparait. Vous pouvez choisir n’importe quelle caractéristique en cliquant sur la case vide qui lui correspond. Cliquer de nouveau sur ladite case pour annuler l’option correspondante.

Options du Tableau de Jeu…

Cette option vous permet de déterminer des caractéristiques particulières pour le jeu. Quand vous sélectionnez Options du Tableau de Jeu, une boîte de dialogue apparait. Pour choisir une caractéristique, cliquez sur sa case vide. Pour annuler une caractéristique, cliquez à nouveau sur cette caractéristique.

Si vous sélectionnez l’option “Faire virevolver le Pion…”, cela fera tournoyer le pion du joueur dont c’est le tour de participer sur le tableau ou dans son secteur de jeu, ou bien les deux à la fois. Si vous choisissez l’option “Dessiner un titre de propriété sur chaque propriété hypothéquée”, il vous sera plus facile de vous rappeler quelles sont les propriétés hypothéquées. Si vous teintez en gris les propriétés vacantes ou non-hypothéquées, il vous sera plus facile durant la partie de les identifier.

Arriere-Plan des Films d’animation…

Cette option vour permet de changer l’arrière-plan des films d’animation qui mettent en scène les mouvements des pions. Quand vous sélectionnez Arrière-Plan des Films d’animation, une boîte de dialogue apparait, d’où vous pouvez choisir un fichier arrière-plan. Sélectionnez ou entrez un nom de fichier arrière-plan et cliquez ensuite sur OK. Le nouvel arrière-plan figurera sur chaque film d’animation des mouvements de pion.

Sauvegarder options comme défaut

Cette option entraînera la sauvegarde de tous les paramètres courants de vos options de jeu, et les classera comme valeurs par défaut. La prochaine fois que vous lancerez le jeu MONOPOLY CD-ROM, ces paramètres seront chargés automatiquement. Si, durant des parties ultérieures, vous changez de nouveau les paramètres, vous devrez sélectionner de nouveau cette option pour les sauvegarder.
