Menu des Fichiers

Nouvelle partie…

Choisissez cette option pour commencer une nouvelle partie de MONOPOLY CD-ROM. Si une partie est déjà en cours, vous aurez la possibilité de sauvegarder cette partie avant d’en commencer une autre.

Ouvrir une partie…

Choisissez cette option pour charger une partie de MONOPOLY CD-ROM que vous aviez précédemment sauvegardée. L’option “Ouvrir une partie” apparait et vous pouvez choisir la partie que vous désirez charger et reprendre.

Sauvegarder une partie…

Choisissez cette option pour sauvegarder une partie qui est en cours. En sauvegardant cette partie vous pourrez la continuer plus tard. Cette option n’est offerte que pour les parties jouées sur un seul ordinateur et non pas pour les parties disputées sur Internet ou sur un réseau. Quand l’option Sauvegarder une partie apparait, créez un nom de fichier pour la partie que vous désirez sauvegarder.

Quitter

Cette option vous permet de quitter le MONOPOLY CD-ROM. Vous aurez la possibilité de sauvegarder la partie en cours.
