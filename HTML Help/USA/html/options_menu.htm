Options Menu

Custom game rules...

The MONOPOLY CD-ROM game plays the traditional rules of the MONOPOLY game by default. The Custom game rules allow you to change certain aspects of the game. To change any of the default options, select Custom game rules. A dialog box appears. Click on any of the options that you wish to change. If you are going to play the MONOPOLY CD-ROM game across the Internet or on a Network, all remote players get a chance to change or agree to any options you have selected. This process continues until all changes have been agreed upon. More information about beginning an Internet or Network game can be found in Starting a Game.

International options...

The MONOPOLY CD-ROM game allows you to compete against players from other countries across the Internet. Because there are slight rules differences in different countries, the International options menu allows you to choose to play with some of these rules differences. Only your standard Chance and Community Chest cards are initially selected for play. Click on one or more country decks to add them to your game. Click on an already selected deck to remove it from your game. To choose any of the rules options, click on the rule to select it. Click on an already selected rule to remove it from the list.

Digital sound effects

Select this option to turn game sound effects on and off. A check mark next to this option indicates that sound effects are currently on.

Background MIDI music...

Selecting this option presents a dialog box that allows you to turn game music on and off, change the MIDI output device, and arrange the sound track in the order that you prefer. A check mark next to this option indicates that game music is currently on.

Movie display options...

Select this option to turn on and off the 3D movies that play in the middle of the game board. Each option indicates which movie or set of movies will be turned off by deselecting it. Since all movies are turned on by default, you must selectively turn off each movie type by clicking on it.

Computer AI options...

This option lets you set some characteristics for the computer players in the game. After selecting Computer AI options, a dialog box will appear. You may turn any feature on by clicking on the empty box next to the feature. You may also deselect by clicking on a feature that is currently selected.

Game board options...

This option lets you set some characteristics for the game. After selecting Board display options, a dialog box will appear. To select a feature, click on its empty box. To deselect a feature, click on a currently selected feature.

Selecting the Spinning Token features will cause the current player's token to rotate on the game board, in the player area, or both during the token's turn. Drawing a deed on mortgaged property will make it easy to remember which properties are mortgaged. Graying out unowned or mortgaged properties all make it easier during the game to identify these properties.

Movie backdrop...

This option allows you to customize the backdrop of the token movement movies. After selecting Movie backdrop, a dialog box appears allowing you to choose a movie backdrop file. Select or type in a backdrop filename, and then click on OK. The backdrop will be visible the next time a token movie plays.

Save options as default

This option will save all of your current game options settings. The next time you start the MONOPOLY CD-ROM Game, these settings will be automatically restored. In future games, if you change the settings again, you'll need to choose this option again to save them.
