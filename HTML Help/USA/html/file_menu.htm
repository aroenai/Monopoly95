File Menu

New game…

Select this option to begin a new game of MONOPOLY CD-ROM. If a game is in progress, you will be offered the chance to save the current game before starting a new one.

Open game…

Select this option to restore a previously saved game of MONOPOLY CD-ROM. The Open dialog will appear, and you may select which saved game to open.

Save game…

Select this option to save a game currently in progress. Saving allows you to continue a game at a later time. This option will only be available for a game played on a single computer. When the Save game dialog appears, enter a filename for the save game.

Exit

This option quits you out of the MONOPOLY CD-ROM game. You'll be given the chance to save a game in progress.
